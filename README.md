# MyLab LIMS v3.0

  نظام إدارة المختبرات الطبية — يعمل على **الويب** و**الأندرويد** و**ويندوز**.

  ## ✅ Google Sign-in

  | المنصة | الطريقة |
  |--------|---------|
  | Android | `@capacitor-firebase/authentication` — Chrome Custom Tabs |
  | Web | `signInWithPopup` — Firebase Auth |
  | Windows | Electron OAuth loopback |

  ## 🚀 التشغيل

  ```bash
  npm install --legacy-peer-deps
  npm run dev           # ويب
  npm run electron:dev  # ويندوز
  npm run android:build # أندرويد APK
  ```
